# Math Quiz Game

A simple math quiz game implemented in C++. Players can choose the level of difficulty and the type of mathematical operations to solve a series of randomly generated questions.

## Features

- Multiple difficulty levels: Easy, Medium, Hard, and Mixed.
- Various mathematical operations: Addition, Subtraction, Multiplication, Division, and Mixed.
- Randomly generated questions based on selected difficulty and operation.
- Tracks correct and incorrect answers.
- Displays final results at the end of the game.

## Usage

Upon running the program, you will be prompted to:

1. Enter the number of questions you want to answer.
2. Select the level of difficulty:
   - [1] Easy
   - [2] Medium
   - [3] Hard
   - [4] Mixed
3. Select the type of operation:
   - [1] Addition
   - [2] Subtraction
   - [3] Multiplication
   - [4] Division
   - [5] Mixed

Answer each question by entering the correct result (Only integer). At the end of the game, your results will be displayed, including the number of correct and incorrect answers.
